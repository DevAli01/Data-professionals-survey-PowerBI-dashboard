📌 Project Introduction

This Power BI dashboard analyses survey responses from 630 data professionals to uncover trends in salary, job roles, programming language preferences, and workplace satisfaction.

The objective is to transform raw survey data into clear, decision-support insights using interactive visualizations and structured data modeling.

🔎 Key Business Questions

Which job roles earn the highest average salaries?

Is there a noticeable salary difference between male and female professionals?

Which programming languages dominate the data industry?

How satisfied are data professionals with their salary and work/life balance?

Which countries have the highest representation in the dataset?

🛠 Tools Used

Power BI Desktop

Power Query (Data Cleaning & Transformation)

Data Modelling (Relationships)

DAX (Measures & KPIs)

Interactive Dashboard Design

📈 Key Insights

Data Scientists report the highest average salary among surveyed roles.

Python is overwhelmingly the most preferred programming language.

Salary satisfaction (~4.3/10) is notably lower than work/life balance satisfaction (~5.7/10).

The United States and India represent a significant portion of survey participants.

The dataset shows a relatively balanced gender salary distribution within roles.
